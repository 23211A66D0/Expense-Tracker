# 💰 Expense Tracker

## Overview

Expense Tracker is a web-based personal finance management application that helps users track their income, expenses, and overall financial status. The application provides a simple and intuitive interface for recording transactions, monitoring spending habits, and maintaining better financial control.

## Features

* Add income and expense transactions
* Track total balance in real-time
* View transaction history
* Categorize financial records
* Delete transactions when needed
* Responsive and user-friendly interface
* Instant balance updates
* Local storage support for data persistence

## Technologies Used

### Frontend

* CSS3
* JavaScript

### Storage

* Browser Local Storage

## How It Works

1. Enter income or expense details.
2. Add the transaction.
3. The application automatically updates:

   * Total Balance
   * Total Income
   * Total Expenses
4. View all transactions in the transaction history section.
5. Remove transactions whenever required.

## Key Functionalities

* Real-time financial tracking
* Expense monitoring
* Income management
* Automatic balance calculation
* Transaction history maintenance
* Data persistence using Local Storage
